# 📊 Sales Data Analysis using Python & Pandas

## 📌 Objective
The goal of this project is to analyze sales data and extract meaningful business insights such as:
- Best selling products
- Highest revenue-generating products
- Most profitable cities
- Average order value

---

## 📂 Dataset
A custom dataset was created containing the following fields:
- Order ID
- Product
- Price
- Quantity
- City

---

## ⚙️ Steps Performed

1. **Data Loading**
   - Created dataset using Pandas DataFrame

2. **Data Transformation**
   - Created a new column:
     - `Revenue = Price × Quantity`

3. **Data Analysis**
   - Identified best selling product using total quantity
   - Found highest revenue product using grouped revenue
   - Determined top-performing city based on revenue
   - Calculated total revenue and average order value

4. **Data Visualization**
   - Line chart for revenue per order
   - Bar chart for product vs quantity
   - Bar chart for city vs revenue

---

## 📊 Key Insights

- **Mobile** is the most demanded product based on quantity sold  
- **Laptop** generates the highest revenue  
- **Lahore** is the most profitable city  
- Average order value indicates potential to increase revenue through bundling or pricing strategies
